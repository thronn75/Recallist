# # FrameShift

FrameShift lets you capture, compare, and restore your entire macOS workspace with precision.

Create scenes of your current window layout, recall them instantly, and understand how your workspace evolves over time through built-in analytics.

---

## ✨ Features

### Scene Capture & Recall
- Capture full multi-display layouts
- Restore window positions and sizes
- Optional confirmation before recall
- Auto sessions (passive layout tracking)

### Compare & Mapping
- Before / After visual comparison
- Window movement mapping with confidence scoring
- Morph preview before confirm
- "No layout changes" intelligent detection

### Insights & Analytics
- Recalls and capture metrics (7d / 30d)
- Average moved windows per recall
- No-changes rate
- Match quality scoring
- Heat Evolution graph (Apple-style scrub interaction)
- Time-of-day heatmap (weekday × hour)
- Top Scenes filtered by time slot

### Premium Thumbnails
- Window-level capture via ScreenCaptureKit
- Smart fallback rendering
- Blurred backdrop + rounded card composition
- Optimized caching for performance

---

## 🔒 Privacy First

FrameShift runs entirely on your Mac.

- No cloud services
- No analytics tracking
- No external servers
- No user data collection

Screen Recording permission is used **only** to generate window thumbnails.
Accessibility permission is required **only** to move and resize windows during recall.

All data is stored locally in your Application Support directory.

---

## 🖥 Requirements

- macOS 14 or later
- Accessibility permission (for window movement)
- Screen Recording permission (optional, for thumbnails)

---

## 🚀 Philosophy

Your workspace is part of your thinking process.

FrameShift treats layout as state.
Not just windows — but context.
